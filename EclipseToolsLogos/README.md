Eclipse Tools Logo Icons and Images
==============================

This directory contains a collection of "Tools Logo" images and icons. 

Contributed in bug 370021 [1] they are primarly intended for those in the Tools Top Level project that would like to use a common "Tools Logo" icon, for various purposes such as their "branding icon" for features represented in the Eclpse "About Box". 

See Eclipse Help [2] and the Eclipse wiki [3] for more information about "branding" and the use of icons.
