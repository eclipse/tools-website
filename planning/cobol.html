Project Name: COBOL

Project Leaders: Ron Langer

Project Description: The COBOL IDE for Eclipse Subproject will build a fully functional COBOL Integrated Development Environment (IDE) for the Eclipse platform. Our focus is COBOL application development on Windows/Solaris/Linux for deployment on each platform. We are interested in participation from others who would like to extend our work in other directions. e.g. wizards for developing applications that use particular library or database or messaging APIs, or extension to other languages.

See http://www.eclipse.org/cobol
